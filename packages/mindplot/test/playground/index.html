WiseMapping Mindplot - Playground

You will find here a set of examples that shows how you can use integrate WiseMapping Mindplot.

  • Mindmap Render: Simple integration to load and render mindaps in read only mode
  • Mindmap Editor: Example on how mindplot can be used for mindmap edition. Browser local storage is used for persistance.
  • Embedded: Example on how to embeded editor in a iframe.
  • Layout Renders: Showcase the different supported layouts.
